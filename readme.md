# build instruction

Node : v16.16.0
Npm / npx : inclus dedans

commandes : 

installer les modules Node requis
`npm install`

démarrer le launcher :
`npm start`

compiler le launcher
```
npm run build-nix # compile pour linux en .appimage
npm run build-win # compile pour windows en .exe
```
